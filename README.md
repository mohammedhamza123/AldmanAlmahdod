# موقع شركة الضمان المحدود

موقع إلكتروني احترافي لشركة الضمان المحدود لاستيراد المواد الغذائية والفواكه الطازجة.

## المميزات

- ✅ تصميم عربي احترافي (RTL)
- ✅ متجاوب مع جميع الأجهزة
- ✅ صفحات متعددة (الرئيسية، من نحن، الخدمات، التواصل، سياسة الخصوصية)
- ✅ واجهة مستخدم عصرية وجذابة
- ✅ سريع وسهل الاستخدام
- ✅ نظام إدارة للرسائل مع قاعدة بيانات SQLite
- ✅ لوحة تحكم للإدمن لعرض الرسائل الواردة

## المتطلبات

- Python 3.7 أو أحدث
- pip

## التثبيت والتشغيل

### 1. إنشاء البيئة الافتراضية (اختياري لكن موصى به)

**في Windows (PowerShell أو CMD):**
```bash
python -m venv venv
```

**تفعيل البيئة الافتراضية:**
```bash
venv\Scripts\activate
```

**في Linux/Mac:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 2. تثبيت المكتبات المطلوبة

```bash
pip install -r requirements.txt
```

### 3. تشغيل الموقع

```bash
python main.py
```

أو باستخدام uvicorn مباشرة:

```bash
uvicorn main:app --reload
```

### 4. فتح الموقع

افتح المتصفح وانتقل إلى:
```
http://localhost:8000
```

## نظام الإدارة

### تسجيل الدخول

للوصول إلى لوحة التحكم:
```
http://localhost:8000/admin/login
```

**بيانات الدخول الافتراضية:**
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

⚠️ **مهم:** يجب تغيير كلمة المرور في الإنتاج!

### لوحة التحكم

بعد تسجيل الدخول، يمكنك:
- عرض جميع الرسائل الواردة من صفحة "تواصل معنا"
- تحديد الرسائل كمقروءة
- حذف الرسائل
- عرض إحصائيات الرسائل

## الصفحات المتاحة

### الصفحات العامة:
- `/` - الصفحة الرئيسية
- `/about` - من نحن
- `/services` - خدماتنا / منتجاتنا
- `/contact` - تواصل معنا
- `/privacy` - سياسة الخصوصية

### صفحات الإدارة:
- `/admin/login` - تسجيل الدخول
- `/admin/dashboard` - لوحة التحكم (تتطلب تسجيل الدخول)
- `/admin/logout` - تسجيل الخروج

## قاعدة البيانات

يتم استخدام SQLite لحفظ:
- **جدول admin:** بيانات المسؤول (اسم المستخدم وكلمة المرور المشفرة)
- **جدول messages:** جميع الرسائل الواردة من صفحة التواصل

قاعدة البيانات يتم إنشاؤها تلقائياً عند أول تشغيل للموقع.

## التخصيص

### تغيير بيانات الشركة

يمكنك تعديل بيانات الشركة في ملف `main.py` في متغير `company_data`:

```python
company_data = {
    "name": "اسم الشركة",
    "address": "العنوان",
    "phone1": "رقم الهاتف الأول",
    "phone2": "رقم الهاتف الثاني"
}
```

### تغيير كلمة مرور المسؤول

يمكنك تغيير كلمة المرور الافتراضية في ملف `database.py` في دالة `init_db()`.

أو يمكنك استخدام Python مباشرة:

```python
import database
import hashlib

# تغيير كلمة المرور
new_password = "كلمة_المرور_الجديدة"
password_hash = hashlib.sha256(new_password.encode()).hexdigest()

# تحديث في قاعدة البيانات
import sqlite3
conn = sqlite3.connect("database.db")
cursor = conn.cursor()
cursor.execute("UPDATE admin SET password_hash = ? WHERE username = 'admin'", (password_hash,))
conn.commit()
conn.close()
```

## الأمان

⚠️ **تحذيرات مهمة:**

1. **تغيير SECRET_KEY:** في ملف `main.py`، قم بتغيير `SECRET_KEY` إلى قيمة عشوائية قوية
2. **تغيير كلمة المرور:** غير كلمة المرور الافتراضية فوراً
3. **HTTPS:** في الإنتاج، استخدم HTTPS لحماية الجلسات
4. **حماية قاعدة البيانات:** تأكد من عدم إتاحة ملف `database.db` للعامة
