# موقع شركة الضمان المحدود

موقع إلكتروني احترافي لشركة الضمان المحدود لاستيراد المواد الغذائية والفواكه الطازجة.

## المميزات

- ✅ تصميم عربي احترافي (RTL)
- ✅ متجاوب مع جميع الأجهزة
- ✅ صفحات متعددة (الرئيسية، من نحن، الخدمات، التواصل، سياسة الخصوصية)
- ✅ واجهة مستخدم عصرية وجذابة
- ✅ سريع وسهل الاستخدام
- ✅ نظام إدارة للرسائل مع قاعدة بيانات SQLite
- ✅ لوحة تحكم للإدمن لعرض الرسائل الواردة

## المتطلبات

- Python 3.7 أو أحدث
- pip

## التثبيت والتشغيل

### 1. إنشاء البيئة الافتراضية (اختياري لكن موصى به)

**في Windows (PowerShell أو CMD):**
```bash
python -m venv venv
```

**تفعيل البيئة الافتراضية:**
```bash
venv\Scripts\activate
```

**في Linux/Mac:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 2. تثبيت المكتبات المطلوبة

```bash
pip install -r requirements.txt
```

### 3. تشغيل الموقع

```bash
python main.py
```

أو باستخدام uvicorn مباشرة:

```bash
uvicorn main:app --reload
```

### 4. فتح الموقع

افتح المتصفح وانتقل إلى:
```
http://localhost:8000
```

## نظام الإدارة

### تسجيل الدخول

للوصول إلى لوحة التحكم:
```
http://localhost:8000/admin/login
```

**بيانات الدخول الافتراضية:**
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

⚠️ **مهم:** يجب تغيير كلمة المرور في الإنتاج!

### لوحة التحكم

بعد تسجيل الدخول، يمكنك:
- عرض جميع الرسائل الواردة من صفحة "تواصل معنا"
- تحديد الرسائل كمقروءة
- حذف الرسائل
- عرض إحصائيات الرسائل

```
wepSite/

├── database.py            # إدارة قاعدة البيانات
├── database.db            # قاعدة البيانات (يتم إنشاؤها تلقائياً)
├── requirements.txt        # المكتبات المطلوبة
├── README.md              # ملف التوثيق
├── templates/             # ملفات HTML
│   ├── base.html         # القالب الأساسي
│   ├── index.html        # الصفحة الرئيسية
│   ├── about.html        # صفحة من نحن
│   ├── services.html     # صفحة الخدمات
│   ├── contact.html      # صفحة التواصل
│   ├── privacy.html      # صفحة سياسة الخصوصية
│   └── admin/            # صفحات الإدارة
│       ├── login.html    # صفحة تسجيل الدخول
│       └── dashboard.html # لوحة التحكم
└── static/               # الملفات الثابتة
    ├── css/
    │   ├── style.css     # ملف التنسيق الرئيسي
    │   └── admin.css     # تنسيق لوحة التحكم
    └── js/
        ├── main.js       # ملف JavaScript الرئيسي
        └── admin.js      # JavaScript للإدارة
```

## الصفحات المتاحة

### الصفحات العامة:
- `/` - الصفحة الرئيسية
- `/about` - من نحن
- `/services` - خدماتنا / منتجاتنا
- `/contact` - تواصل معنا
- `/privacy` - سياسة الخصوصية

### صفحات الإدارة:
- `/admin/login` - تسجيل الدخول
- `/admin/dashboard` - لوحة التحكم (تتطلب تسجيل الدخول)
- `/admin/logout` - تسجيل الخروج
