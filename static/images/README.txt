===========================================
  إضافة الشعار (Logo)
===========================================

يرجى وضع ملف الشعار هنا باسم:
  logo.png

المواصفات الموصى بها:
  - الصيغة: PNG (شفاف الخلفية أفضل)
  - الحجم: 500x500 بكسل أو أكبر
  - الدقة: عالية الجودة

بعد وضع الملف، سيظهر الشعار تلقائياً في:
  ✓ شريط التنقل (Navbar)
  ✓ الصفحة الرئيسية (Hero Section)
  ✓ التذييل (Footer)
